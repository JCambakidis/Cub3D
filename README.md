Your mission, kill all enemies !!!

Inputs :

- Move : ```W, A, S, D```
- Run : ```Tab Left```
- Rotate Camera : ```←	↑ →	↓```
- Attack : ```Alt Left```
- Jump : ```Space```
- Guard : ```Tab Right```
- Show Mini-Map : ```Shift```

Debug Mode Inputs :

- Activating Debug : ```<```
- Activate Fly Mode : ```end```
- Fly : ```Space```
- Change FOV : ```*, /```
- Change Render Distance : ```+, -```
- Change Pixels Resolution : ```PageUP, PageDown```

In Cube3d Folder use : 
```bash
./cub3d maps/map.cub
```
